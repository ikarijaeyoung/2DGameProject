# 2DGameProject
My Game
나의 구현 계획
장르 : 방치형게임
메인화면 - 캐릭터 홈 화면 - 자동 전투 - 보스 - 죽으면 다시 캐릭터 홈 화면

우선적으로 구현하고싶은 기능 :
자동 전투: 
입력받아 전투를 할 수는 없음.
캐릭터의 공격 사거리에 Monster가 들어오면 공격 속도만큼 공격 시작.
전투에서 얻은 재화로 캐릭터를 강력하게 할 수 있음.
캐릭터를 강력하게 하는 기능은, 캐릭터 홈 화면에서 할 것.

몬스터 공격 패턴 :
몬스터는 기본적으로 비활성화 되어있음.(또는 움직이지 않음.)
가장 플레이어와 가까운 몬스터 1마리만 움직이도록.
몬스터도 마찬가지로 공격범위 내에 Player가 들어오면 공격 속도만큼 공격 시작.

저장 기능 (Json) :
게임을 껏다 다시 실행하여도 이전에 진행했던것이 남아있어야함.

캐릭터 홈 화면 :
전투에서 얻은 재화로 캐릭터의 공격력, 공격속도, 체력, 방어력을 향상시킬 수 있음.
전투에서도 향상된 스텟으로 전투하게 될 것.

죽으면 다시 홈 화면으로 돌아가기 :
Player Hp가 0이하가 되면 죽음,
죽으면 캐릭터 홈 화면에서 스폰.

타이틀 화면 : 
새로운 게임, 이어하기, 종료하기 기능
게임 음량 설정
